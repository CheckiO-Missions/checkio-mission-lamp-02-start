Це друга місія в серії Lightbulb. Кожне наступне завдання я постараюся трохи ускладнити.

Ти вже навчився рахувати, скільки часу світила лампочка, або як довго кімната була освітлена. Тепер додамо ще один параметр - час початку підрахунку.

Це означає, що світло продовжує вмикатися та вимикатися, як і раніше. Але тепер, в результаті роботи функції, я хочу знати не тільки скільки часу було світло в кімнаті, але і скільки часу кімната була освітлена, починаючи з певного моменту.

Додається ще один аргумент – start_watching, а якщо його не передано, рахуємо як у попередній версії програми за весь період.

Вхідні дані: Два аргументи і лише перший потрібний. Перший — це {% if is_js %}масив об’єктів Date{% else %}список об’єктів datetime{% endif %}, а другий - {% if is_js %}об’єкт Date{% else %}об'єкт datetime{% endif %}.

Вихідні дані: Кількість секунд як ціле число.

Приклади:

{% if is_js %} 
sumLight([
    new Date(2015, 1, 12, 10, 0, 0),
    new Date(2015, 1, 12, 10, 0, 10),
],
new Date(2015, 1, 12, 10, 0, 5)) == 5

sumLight([
    new Date(2015, 1, 12, 10, 0, 0),
    new Date(2015, 1, 12, 10, 0, 10),
], new Date(2015, 1, 12, 10, 0, 0)) == 10

sumLight([
    new Date(2015, 1, 12, 10, 0, 0),
    new Date(2015, 1, 12, 10, 10, 10),
    new Date(2015, 1, 12, 11, 0, 0),
    new Date(2015, 1, 12, 11, 10, 10),
], new Date(2015, 1, 12, 11, 0, 0)) == 610
{% else %} 
sum_light([
    datetime(2015, 1, 12, 10, 0, 0),
    datetime(2015, 1, 12, 10, 0, 10),
],
datetime(2015, 1, 12, 10, 0, 5)) == 5

sum_light([
    datetime(2015, 1, 12, 10, 0, 0),
    datetime(2015, 1, 12, 10, 0, 10),
], datetime(2015, 1, 12, 10, 0, 0)) == 10

sum_light([
    datetime(2015, 1, 12, 10, 0, 0),
    datetime(2015, 1, 12, 10, 10, 10),
    datetime(2015, 1, 12, 11, 0, 0),
    datetime(2015, 1, 12, 11, 10, 10),
], datetime(2015, 1, 12, 11, 0, 0)) == 610
{% endif %}

Передумова:

  • Масив натискання кнопки завжди сортується за зростанням
  • Масив натискання кнопки не має повторюваних елементів
  • Кількість елементів завжди рівномірна (світло з часом вимикається)
  • Мінімальна можлива дата - 01.01.1970
  • Максимальна можлива дата 9999-12-31
